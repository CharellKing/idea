# AI产品描述生成器

## 应用概述
一个使用人工智能自动生成高质量产品描述的Shopify应用，帮助商家节省时间并提高内容质量。

## 核心功能
- 一键生成产品描述
- 多种语调和风格选择（专业、休闲、豪华等）
- SEO优化关键词整合
- 多语言支持
- 批量生成功能

## 目标用户
- 产品众多的商家
- 非英语母语的商家
- 希望提高SEO表现的商家
- 时间紧张的中小型企业
